## What is String Art Generator?

String Art Generator is a free, browser-based tool that converts any photograph into a string art pattern. Upload an image, adjust a few parameters, and the generator computes the optimal sequence of pin-to-pin thread paths needed to recreate that image as a physical string art piece.

Everything runs client-side — no data is sent to a server, no installation is required.

## Key features

- **No installation** — runs entirely in the browser using OpenCV.js and NumJS
- **Configurable output** — adjust the number of pins, lines, and line weight to match your physical hoop
- **Pin sequence export** — save the generated sequence as a list of numbers to recreate your pattern later
- **Step-by-step assistant** — an interactive guide walks you through stringing each line one at a time
- **Example patterns** — three built-in sequences let you preview the assistant without generating from an image
- **Real-time preview** — the canvas updates every 10 lines so you can watch the pattern emerge

## Live tool

The generator is hosted at [halfmonty.github.io/StringArtGenerator](https://halfmonty.github.io/StringArtGenerator/). Open it in any modern browser — no account or sign-up needed.

<Note>
  The page uses WebAssembly via OpenCV.js, which can be large to load. Wait for the **"Generator is ready."** status message before uploading an image.
</Note>

## How string art works

A circular wooden or embroidery hoop is fitted with evenly spaced nails (pins) around its perimeter. A single continuous thread is wound between pins in a specific sequence. When viewed from a distance, the overlapping threads form the illusion of a photographic image.

The generator automates the hardest part — figuring out the optimal pin sequence — so you can focus on the physical craft.
